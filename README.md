# Calculator Hub

A production-grade, offline-capable calculator web application built with pure HTML, CSS, and JavaScript.

## Included Calculators

- Basic Calculator
- Scientific Calculator
- BMI Calculator
- Loan / EMI Calculator
- Percentage Calculator
- Unit Converter

## Features

- Fully responsive
- Dark mode
- SEO optimized
- Schema markup
- Accessibility compliant
- No frameworks
- GitHub Pages ready

## Deployment

1. Clone repository
2. Push to GitHub
3. Enable GitHub Pages (root)
